**ID:** 04.001| Multi-configuration project Configuration > Enable or Disable the Project

**Description:**
**As a** user,
**I want to** enable or disable a Multi-configuration project directly from its configuration page,
**So that I can** temporarily prevent the project from running builds without deleting it or modifying its configuration.

**Preconditions**
1. Jenkins instance is up and running and accessible to the user.
2. User is authenticated and has permissions to configure jobs.
3. User has already created a **Multi-configuration project** and is on the configuration page.

**Acceptance Criteria:**

1. While on the configuration page at the top of the page on the right side you can see an toggle switch to enable or disable the current project.
2. The user can:
- toggle the **Disabled** switch to disable the project.
- toggle the **Enabled** switch to enable the project.
- сlick **Save** after toggling the switch to disable or enable the project.
3. When the project is disabled, the Multi-configuration project page shows `This project is currently disabled` along with an Enable button. After clicking Enable, the project page is refreshed automatically and the warning message disappears.
