**ID:** US_09.004  | Build history>  Handling an Empty Build History

**Description:**

**As a** user,
**I want to** view the Build History section when no builds have been executed for the current item,
**So that I can** understand that no builds exist yet

**Acceptance Criteria:**

1. On the Main page of the Jenkins dashboard, clicking the **Build History** link in the left navigation menu redirects the user to the **Build History** page, which displays an empty table with the column headers: S, Build, Time since, Status. 

2. While on the **Build History** section with empty state, the user can toggle icon sizes (S, M, L) to display build status icons in Small, Medium, or Large format.
