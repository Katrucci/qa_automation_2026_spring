# User Story: US_01.001 | New Item > Select an Item Type

## Description

**As a** user,  
**I want to** select an item type (Freestyle project, Pipeline, Multi-configuration project, etc.) when creating a new Jenkins item,  
**So that** define the appropriate project type for my CI/CD workflow needs before proceeding with configuration.

---

## Preconditions

1. Jenkins instance is up and running and accessible to the user
2. User is authenticated and has permissions to create new items. 
3.  User has navigated to the New Item creation page (reached via "New Item" link on the dashboard — note: there's a "+" icon next to the "New Item" label.) 
4. Entered the unique folder name.

---
**Acceptance Criteria:**
1. By clicking the "New Item" (with a "+" icon) link on the left navigation menu of the Jenkins dashboard,
the user is redirected to the item creation page (http://localhost:8080/view/all/newJob), where the following item types are available as selectable options:
- Pipeline
- Freestyle project
- Multi-configuration project
- Folder
- Multibranch Pipeline
- Organization Folder

2. While on the item creation page, the user can:
- Enter a valid and unique item name (alphanumeric, hyphen, underscore — no spaces or special characters).
- Select one item type from the available options.
- Click "OK" only after both a valid name and an item type are provided.
- Return to the Jenkins dashboard without creating an item — by clicking the "Jenkins"  text with the logo in the top-left corner.
3. After the user enters a valid unique item name, selects an item type, and clicks "OK",  the user is redirected to item configuration page.
